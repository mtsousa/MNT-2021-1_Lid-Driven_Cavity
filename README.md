## Configuração

### Ambiente de desenvolvimento

1. Crie o ambiente

```bash
python -m venv mnt_venv
```
2. Ative o ambiente

    - Em ambiente Windows (git bash)
    ```bash
    source mnt_venv/Scripts/activate
    ```
    
    - Em ambiente Linux
    ```bash
    source mnt_venv/bin/activate
    ```

3.  Instale as depêndencias

```bash
pip install -r requirements.txt
```
